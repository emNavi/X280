##

该项目开发了无人机科研平台 **X280**、 **152b** 等，精简并集成了多种基于无人机平台的前沿研究算法。本仓库是 emNavi 项目基于 **X280** 型无人机的二次开发平台，以及几种典型智能算法的一键部署。 - [使用文档 https://emnavi.tech/](https://emnavi.tech/).
